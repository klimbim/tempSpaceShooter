# Mission Background AI Brief

## Purpose

This document describes how to create background art
for missions in the space shooter game.

## Core Rules

- One mission = one background pack
- Backgrounds are layered (parallax)
- No baked lighting or shadows
- Designed for long play sessions

## Folder Structure

Each mission background contains:

- layers/
- tiles/
- objects/
- config.json

## Style Guidelines

- Top-down view
- Subtle color palette
- No visual noise
- Must not compete with gameplay readability

## Technical Constraints

- Tileable where specified
- PNG, transparent background
- Designed for scaling and tinting
